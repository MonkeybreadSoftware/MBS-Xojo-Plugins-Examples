This is a small utility to help manageing the Monkeybread Software Realbasic Plugins.

Installation

Way one: Copy the compiled application into the Realbasic folder. Way two: Make an alias named "Plugins" to your plugins folder into the application folder of MBS Plugin Manager so it can find the plugins.

Version history

  • Version 2.0 accepts now .rbx plugin files.
  • Version 3.0 was updated to all the changes in plugins in the last year and works with Real Studio 2012.

Limitations

Does not work while Realbasic is running.

It is possible that the plugins depend on a certain load order Realbasic may report errors. CF plugin part must be loaded first in some plugin versions.
