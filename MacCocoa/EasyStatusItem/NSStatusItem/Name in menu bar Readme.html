About Name in menu bar 1.0

This small application does nothing, but showing the current name of the user.

It is an example on what you can do in Realbasic using the MBS Plugins on global menus.

This application runs in the background and can be quit using the Activity Monitor (see /Applications/Utilities folder).

Installation

Drop the application wherever you want it to be. You only need the application file and if you like you can add it to the login items in your user settings in system preferences to have it launched on login time.
