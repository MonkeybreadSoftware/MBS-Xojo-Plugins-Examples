About the DynaPDF license

For the dynapdf library exists a license which is included with the examples.

The plugin license code can be purcharsed from Monkeybread Software.

DynaForms and Monkeybread Software do have an agreement that Monkeybread Software is allowed to sublicense the library as part of the plugin. Still you are not allowed by to sublicense the library or the plugin.

If you are worried about one of the paragraphs in the license, please contact DynaForms directly. You may be able to get an agreement for some special software usages.
