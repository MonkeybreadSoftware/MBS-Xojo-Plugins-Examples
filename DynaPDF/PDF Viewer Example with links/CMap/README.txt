The files in this folder contain the predefined CMaps for PDF 1.7 as well as
several additional CMaps provided by Adobe Systems.

The path to external CMaps can be set with SetCMapDir(). External CMaps must
be available when extracting text from Asian PDF files.

When including these CMaps in your application please visit from time to time
Adobe's website to download the latest versions:

http://opensource.adobe.com/wiki/display//pdfmapping/Mapping+Resources+for+PDF
